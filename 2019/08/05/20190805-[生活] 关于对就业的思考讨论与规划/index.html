关于对就业的思考讨论与规划

发表于 2019-08-05 | 更新于: 2019-08-13   |   更新于 2019-08-13 | 分类于 生活 |
字数统计: 1.5k 字 | 阅读时长 ≈ 4 分钟

摘要：”Skilled but jobless”，20190803期的《经济学人》一篇文章直指中国大学毕业生就业难的问题。经济转型，产业革新，世界局势冲突升级，不可避免地给中国和世界经济带来了巨大的冲击，而影响最为直接的便是就业。什么样的工作才是好工作，自己的专业以后前景究竟如何，转行是否可行且有必要，无数的问题困扰着我，也无时不刻萦绕在迷茫的年轻人的心头。在与同学朋友的讨论中，一点思考，记录一下聊以自慰。

怎样的才算好工作

前段时间一直是觉得自己专业很不好，觉得毕业后主流的几个就业方向都不是我想要的，工作也不具有创新性，所以一直想转行。恰巧前段时间和几个其他专业的同学吃饭，也与同学和实习回来的朋友聊了聊，有了一些新的收获。

很多人觉得专业不好，认为自己的专业完全没有核心技术，而且传统市场的行情差，改革创新难度高。

其实对于传统行业来说，自己学起来可能觉得没有核心技术，但是在外人看来还是有很高的行业门槛的，相比互联网来说门槛高的多。传统行业的优势就在于，他有自己独占性的广阔市场。

再者我们自己觉得专业没有创新，是重复性工作，其实现在最火的互联网也未必就处处都是创新，具体投身其中，反而更可能都是调参数、重复别人的代码和工作等等，只是因为他是新东西，外人看来很高大上而已。而自认为没有什么核心技术的专业，也总会有一些大难题，总还是有人攻克了的呀，创新是现代青年必备的思维，但绝不应该成为找工作眼高手低的借口。

撇开以上对职业的创新性的思考，衡量好工作最直接的标准就是薪酬了。现在互联网公司似乎轻轻松松几十万，但是也要看到那些公司很多标配游泳池健身房，其实他们自己也知道员工压力大，很多时候钱多就意味着没有自己的生活。也自己专业的工作也不到一定就那么差，每月万元，955双休，可以有自己的生活，这也未必就不是好工作。

未踏入社会时，满脑子想的都是赚钱，实习后，看着前辈们每天所有时间基本熬在工作上，完全没有个人的生活，才开始思考自己想要的工作到底是什么样的。其实，有的人喜欢工作生活平衡，有的人喜欢与天地奋斗有所收获，好工作的标准从来都只在咱们自己心里。

转行热门可以得到更好的工作？

《数学之美》的作者吴军的一篇文章说：“将来，技术和产业需要结合起来，现有产业+新技术=新产业。每个人要发挥自己的特长，而不是顶着5G的风口，比如一个会计师跨行再学计算机。要理解本质，不要太纠结现象，这很重要。一个会计师利用好计算机，最大地发挥自己的行业专长才是正道”。

现在很多专业的学生都准备了跨专业去做现在最火，也是门槛最低的互联网，当然这也是我之前的想法。但是和同学聊得时候，他以自己的经验非常辩证的来看，这样的选择除了艰辛不说，还有其两面性：

  • 好处是：
    • 互联网在风口，钱多；
    • 还是钱多，在大城市，工作比较好找；
  • 坏处：
    • 如果往软件开发转，对代码能力要求太高，而且还需要了解计算机原理、数据结构等等，相比科班出身完全没有优势，不推荐；
    • 如果往算法转，需要学习和推导大量的基础算法和机器学习算法，也是需要大量准备的，面试可能就是现场手推支持向量机等景点算法；
    • 互联网行业更新太快，新员工的工资比老员工的工资高是非常常见的现象。而且现在火不能保证以后火，一直追着时代的热点跑是跑不过时代的；

反之，也有很多人的观点就是，传统行业虽然不火了，但是毕竟是传统行业，工作是有保障的，而且传统行业相对互联网来说，随着工作年限的增加，自己的价值和经验是会增长的。

现阶段立个Flag

我之前一直觉得自己是准备转行的，去大厂赚大钱，至少是去那种自己专业和新技术结合的创业公司。现在我觉得就是先把本专业作为根据地，尽量让自己有能力进比较好的本专业的企业，然后在自己课题内多学算法和编程这些新的东西，要是以后传统行业也能往智能化方向发展，或许可以有一些能力上的优势呢。如果想转互联网，要是决心不够努力不够，到时候失去了本专业的立足点，互联网那边的技术也不过硬，到时候就是两不沾了，此中权衡，实属不易。
