VPS网络中继及加速

发表于 2019-08-04   |   更新于 2020-02-26 | 分类于 Tutorial |
字数统计: 2.2k 字 | 阅读时长 ≈ 9 分钟

摘要：VPS作为虚拟个人主机，在建站、网络代理等方面具有重要作用。利用VPS搭建个人代理服务器，其原理就是购买境外虚拟个人主机VPS，然后使用加密通信软件建立境内客户端与该境外主机之间的通信，将境内需要访问网站的请求转发给境外主机，由境外主机下载数据后回传，从而实现网络中继的效果。

VPS网络中继及加速

VPS主机选择与购买

首先需要选择一个境外的VPS提供商，老左博客里面有时候会有推荐，其中一些VPS提供商会有不同程度的新人优惠，注意薅羊毛。注册后需要充值一定金额才能领取优惠，部分VPS提供商支持支付宝，有的只能用双币信用卡或paypal支付。

注册完成后在其网站购买VPS，选择最低配置的主机即可。价格因供应商而异，“搬瓦工”这个主机提供商可能稍微便宜点，但是据说用的人多速度不够好。其他的linode、Digitalocean等都可以尝试和比较。总体来说价格一般是5美元/月。

购买VPS的过程中需要选择主机的相关信息，主机机房这个最好通过测速确定，不是离得越近越好，比如部分商家的新加坡机房到大陆是绕道欧洲的，延时较高，还不如美国本土的机房。然后操作系统一般选择Ubuntu16.04×64，由于VPS和我们的电脑之间一般通过SSH连接，所以需要安装putty软件，利用putty gen程序生成一对SSH的公钥和私钥，公钥上传到VPS主机端，私钥自己留着后续通过putty或其他软件验证登录。

与VPS远程通信配置

与VPS主机的通信除了用上述的putty，我更推荐使用MobaXterm，这个软件除了支持命令行会话，还支持ftp文件传输等其他功能，比putty强大的多。

通过设置IP地址、接入端口、私钥、登录名等信息后，建立与VPS主机的会话，通过命令行操控远端的Linux系统，安装对应的软件，编辑配置文件等。

客户端的使用

在V2ray的客户端或者shadowsocks的客户端配置完成后，对于电脑来说，还需要在使用Chrome或Firefox浏览器，安装SwitchySharp插件，在其中配置科学上网的选项，指定本地监测端口和IP地址，需要科学上网时切换到该选项，不需要科学上网时选择直连。

对于手机用户来说，下载shadowsocks并修改相关信息即可使用，较为简单。
